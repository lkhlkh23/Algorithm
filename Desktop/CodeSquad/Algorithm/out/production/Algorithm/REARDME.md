** Text Editor **

1. 입력
    - 줄단위 입력(Create) (List<String>)

2. 출력
    - 줄번호 출력
    - 입력 내용 출력 (Retrieve)
3. 수정
    - 명령어(Update), 커서위치(x, y), (대체문자열)
4. 삭제
    - 명령어(Delete), 커서위치, (all, line, num)

Option) Ctrl + z

"DELETE_ALL"

"DLEETE_LINE" "3"

"DELETE_BACKSPACE" "3"

"PRINT"
